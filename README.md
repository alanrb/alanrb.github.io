# alanrb.github.io

Personal site for **Tuong Nguyen** — software developer working across front-end, backend, and serverless.

Live at https://alanrb.github.io/

## Stack

A single static `index.html`. No build step, no framework, no Bootstrap. Fonts (Fraunces, JetBrains Mono) load from Google Fonts; everything else is inline.

## Running locally

Open `index.html` directly, or serve the folder:

```sh
python3 -m http.server 8000
```

Then visit http://localhost:8000.
